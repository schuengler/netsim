
### Components

- **omnetpp-6.4.0/**  
  OMNeT++ simulation environment (core simulator + IDE + libraries)

- **inet/**  
  INET Framework providing protocol models (Ethernet, IP, ARP, etc.)

- **src/**  
  Custom simulation code

---

## Prerequisites

### OMNeT++

This project is based on the **OMNeT++ 6.4 simulation framework**.

- Installation guide:  
https://doc.omnetpp.org/omnetpp/InstallGuide.pdf

OMNeT++ provides:
- Simulation kernel
- Qtenv GUI environment
- Build system (opp_makemake, opp_run)
- Library management for models

---

### INET Framework

This project uses the **INET Framework** for network protocol modeling.

- Documentation & setup guide:  
https://inet.omnetpp.org/Documentation.html

INET provides:
- Ethernet, IP, ARP implementations
- Network devices (hosts, switches, routers)
- Packet and protocol models
